# Deployment

> **Information**
>
> **Stimulsoft Ultimate** package contains libraries, scripts, and other files necessary for working with reports and dashboards in WinForms, ASP.NET, ASP.NET MVC, .NET Core, WPF, JavaScript, PHP, and Java applications. The product includes a report and data analysis engine, reports and dashboards designers and viewers for all supported platforms.

This section describes the various distribution options for reports, libraries, and files supplied with Stimulsoft software.


| **Name** | **Description** |
| --- | --- |
| [Reports.WEB](DLLs_in_Reports_Web_Package.md) | The package contains libraries for working with reports in ASP.NET, ASP.NET MVC, and .NET Core applications. |
| [Reports.NET](DLLs_in_Reports_Net_Package.md) | The package contains libraries for working with reports in WinForms applications. |
| [Reports.JS](Files_in_Reports_JS_Package.md) | The package contains scripts for working with reports in JavaScript applications. |
| [Reports.WPF](DLLs_in_Reports_Wpf_Package.md) | The package contains libraries for working with reports in WPF applications. |
| [Reports.PHP](Files_in_Reports_PHP_Package.md) | The package contains scripts and styles for working with reports in client-server applications using PHP. |
| [Reports.JAVA](JAR_Files_in_Reports_Java_Package.md) | The package contains JAR files for working with reports in Java applications. |
| [Dashboards.WEB](DLLs_in_Dashboards_WEB_Package.md) | The package contains libraries for working with dashboards and reports in ASP.NET, ASP.NET MVC, and .NET Core applications. |
| [Dashboards.WIN](DLLs_in_Dashboards_WIN_Package.md) | The package contains libraries for working with dashboards and reports in WinForms and WPF applications. |
| [Dashboards.JS](Scripts_in_Dashboards_JS_Package.md) | The package contains scripts for working with dashboards and reports in JavaScript applications. |
| [Stimulsoft Designer](Files_in_Stimulsoft_Designer_App.md) | The application contains files, libraries to run the standalone report designer. |
| [Stimulsoft Designer.JS](Files_in_Stimulsoft_Designer_JS_App.md) | The application contains files, scripts to run the standalone JavaScript report designer. |
| [Stimulsoft Demo](Files_in_Stimulsoft_Demo_App.md) | The application contains files, libraries to run the reports and dashboards demonstration. |


> **Information**
>
> A dmg is supplied in which **Stimulsoft Designer for Mac** is presented as a single application. After installation, run the **Stimulsoft Designer.app** file to start the report designer.
