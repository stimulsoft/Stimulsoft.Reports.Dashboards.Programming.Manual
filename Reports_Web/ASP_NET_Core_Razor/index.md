# Reports and Dashboards for ASP.NET Core Razor

**.NET Core** is a cross-platform technology for creating Web applications for Windows, Linux, and macOS. **ASP.NET Core Razor Pages** is a technology that is similar to the Model-View-Controller (MVC) system. **Razor Pages** allow you to create pages with code that can process various requests. This technology makes the creation of scripts for pages easier and more efficient in comparison with the use of the MVC solution. [Stimulsoft](https://www.stimulsoft.com/en) company provides tools for creating, displaying, and transforming reports and dashboards using this technology.
