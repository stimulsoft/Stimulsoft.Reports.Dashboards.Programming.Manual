# Designer

The report designer for **UWP** is designed for creating and editing reports in **Windows** and works with **ARM** and Intel versions. The UI of the designer provides for user convenient control, wide range of tools, components and instruments for creating reports, visual design and previewing. The operations in the designer can be done using touch (hand, stylus) or mouse. The report designer supports Ribbon interface 2013.
