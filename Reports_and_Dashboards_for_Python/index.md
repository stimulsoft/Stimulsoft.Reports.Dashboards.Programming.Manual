# Reports and Dashboards for Python

[Stimulsoft Reports.PYTHON](https://www.stimulsoft.com/en/products/reports-python) and [Stimulsoft Dashboards.PYTHON](https://www.stimulsoft.com/en/products/dashboards-python) comprise a combination of a JavaScript and a Python scripts, along with all the necessary functionality for their interaction. The report is built and exported on the client side using JavaScript, while the Python server side contains everything needed to work with report files and communicate with various SQL data sources. Communication between the client and the server is carried out through AJAX requests that transmit and receive JSON data in a specific format. To enhance the usability of the product, special events and functions have been introduced both on the JavaScript client side and on the Python server side.


The product includes the [report engine](Engine/index.md) itself and the [data analysis](Engine/index.md) core, as well as the [viewer](Web_Viewer/index.md) and report [designer](Web_Designer/index.md) components.
